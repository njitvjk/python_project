Keywords and Identifiers

Keywords are the reserved words in Python.They cannot be used as variables or function or any other identifiers.In Python, keywords are case sensitive.

All the keywords except True, False and None are in lowercase and they must be written as they are. The image shows the list of all keywords.

An identifier is a name given to entities like class, functions, variables, etc. An identifier cannot start with a digit.

Tuples

Tuples look just like list expect you use parenthesis instead of square brackets. Although you cannot modify a tuple, you can assign a new set of values to a variable(reassignment)
